&#x20;🚀 TruthLens AI – Scam Detection Platform



An AI-powered full-stack web application that detects scams, fake job offers, and suspicious messages using Machine Learning and rule-based analysis.



\---



🔍 Overview



TruthLens AI analyzes job posts, emails, and messages to determine whether they are \*\*Safe\*\*, \*\*Medium Risk\*\*, or \*\*Scam\*\*.



It combines:

\- 🧠 Machine Learning (TF-IDF + Logistic Regression)

\- ⚙️ Rule-based keyword detection

\- 📊 Risk scoring system

\- 🎯 Explainable AI output (reasons + confidence)



\---



✨ Features



\- 🔎 Analyze suspicious content in real-time

\- 📊 Risk Score + AI Probability + Verdict

\- 🧠 ML-based classification with confidence score

\- ⚠️ Keyword-based scam detection (UPI, fee, urgent, etc.)

\- 📝 Highlight suspicious words in input

\- 📄 Download PDF report

\- 📁 Save and view analysis history

\- 🗑️ Delete reports from history

\- 🎨 Modern UI with animations and dashboard

\- 🔐 Authentication (Login/Signup)



\---



🧠 ML Pipeline



1\. Data Collection (custom dataset)

2\. Preprocessing (lowercase, stopword removal)

3\. Feature Extraction using TF-IDF

4\. Model: Logistic Regression (multi-class)

5\. Prediction:

&#x20;  - Safe (0)

&#x20;  - Medium Risk (1)

&#x20;  - Scam (2)

6\. Probability → Confidence score

7\. Combined with rule-based scoring
